## 📖 Sobre o Projeto

O **VittaRun Frontend** é a interface web da plataforma de delivery saudável, desenvolvida para proporcionar uma experiência intuitiva e moderna aos usuários.

A aplicação permite visualizar lojas, navegar entre categorias, buscar estabelecimentos e interagir com a API backend do projeto.

---

## ✨ Funcionalidades

| Funcionalidade | Descrição |
|----------------|-----------|
| 🏪 Listagem de lojas | Exibe todas as lojas cadastradas |
| 🔍 Busca de lojas | Pesquisa por nome ou categoria |
| 📱 Interface responsiva | Compatível com desktop e mobile |
| 🧭 Navegação SPA | Rotas dinâmicas com React Router |
| 🎨 Estilização moderna | Interface construída com TailwindCSS |
| 🔗 Integração com API | Comunicação com backend via HTTP |
